📊 SentiTrade Analytics: Bitcoin Market Sentiment & Trader Performance Analysis
Overview

SentiTrade Analytics is a data analytics project that explores the relationship between Bitcoin market sentiment and cryptocurrency trader performance. The project combines historical trading data from Hyperliquid with the Bitcoin Fear & Greed Index to understand how market sentiment influences trading behavior, profitability, and decision-making.

The analysis includes data cleaning, preprocessing, feature engineering, dataset integration, exploratory data analysis (EDA), and loop-based comparative analysis. Various visualizations are used to identify trends in trader performance, coin-wise profitability, trading activity, and market sentiment.

Objectives
Analyze the impact of Bitcoin market sentiment on trader performance.
Compare profitability across Fear and Greed market conditions.
Identify profitable trading patterns and trader behavior.
Perform coin-wise and time-based trading analysis.
Generate actionable insights using data visualization and statistical summaries.
